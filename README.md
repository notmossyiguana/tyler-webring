# the webring for tyler, the creator fans

**Want to join?**
You can either:

**Option 1 — GitHub**

Make a pull request to merge `members.json`, and copy this template in:

```
{
   "name": example
   "site": https://example.com
   "blurb": This is a blurb example
   "button": https://example.com/examplebutton.png (or leave blank)
}
```

**Option 2 — Email**

Send an email to **tylerwebring [at] duck [dot] com**
with the subject line: **"Application"**

Your email should include:

* Your slug/name or alias
* Your site name
* A short blurb describing your website (1–2 sentences)
* Your website’s button (optional)
